## Getting Started

### Local Installation

1. Clone the repo

      git clone <https://git.istad.co/cstad-foundation-2nd-generation/stem-tutors/stem-tutors-website>
  
2. Run the project with live server Open

      Open <http://localhost:5173> with your browser to see the result
